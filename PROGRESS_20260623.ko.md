## 진행 타임라인

### Phase-1: Fable-5 Agentic Foundation ✅ (2026-06-23 20:58 완료)

- **베이스:** `LLM-OS-Models/LFM2.5-8B-A1B-Terminal-ToolBench-Full-SFT-1Epoch` (기존 ToolBench SFT)
- **데이터:** [Glint-Research/Fable-5-traces](https://huggingface.co/datasets/Glint-Research/Fable-5-traces) 4,665 rows → 4,047 rows (618 short-context drop)
- **하이퍼파라미터:**
  - max_seq 8192, epochs 3, LR 5e-7 constant
  - batch 2 × accum 4 × 8 H200 = global 64
  - bf16, FSDP full_shard, Lfm2MoeDecoderLayer auto_wrap, activation_checkpointing
- **결과:** final train_loss 1.277, train_runtime 831초 (≈14분), global_step 192
- **아티팩트:** `/home/work/.data/harness1/models/LFM2.5-8B-A1B__Terminal-ToolBench-Fable5-Agentic-FullSFT-20260623/final_model` (16GB)
- **HF:** `LLM-OS-Models/LFM2.5-8B-A1B-Terminal-Fable5-Agentic` → (예정) `LLM-OS-Models/Fabliq-8B-Agent`

### Phase-2: Reasoning Expansion 🔄 (2026-06-23 21:23 진행 중)

- **베이스:** Phase-1 final_model (Fable-5 agentic)
- **데이터:** WithinUs 135 rows + Helio 146 rows = 281 rows
- **하이퍼파라미터:**
  - max_seq 8192, epochs 4, LR 3e-7 constant (데이터 작고 모델 정제됐으니 낮춤)
  - batch 2 × accum 4 × 8 H200 = global 64
- **목표:** Fable-5 agentic 능력 유지 + general/deep reasoning 추가 (Qwythos 2-phase curriculum 패턴)
- **아티팩트 (예정):** `LFM2.5-8B-A1B__Terminal-Fable5-Phase2-Reasoning-FullSFT-20260623`
- **HF (예정):** `LLM-OS-Models/Fabliq-8B-Agent-Reasoning` 또는 `Fabliq-8B-Plus`

### Phase-3: Agentic Scale-Out (계획)

- **베이스:** Phase-2 final_model
- **데이터:** [Complete-FABLE.5-traces-2M](https://huggingface.co/datasets/...) 2M raw events → 세션 그룹화 후 추출 (수만~십수만 row 예상)
- **이슈:** 2M 데이터가 Glint 형식(context/cot/output)이 아니라 raw Claude Code session events (queue-operation/message/attachment 등). 멀티턴 대화로 재구성하는 전처리 필요.
- **목표:** agentic trajectory 수를 4K → 50K+로 확대 (Qwythos 500M tokens 규모 지향)
- **스크립트 (예정):** `fable_distillation/build_fable5_2m_to_lfm_sft.py`
